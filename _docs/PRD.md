# 업무 스케줄 관리 서비스 PRD

## 프로젝트 목표

장기간 진행되는 업무 프로젝트와 주요 일정을 한눈에 파악하고, 필요한 일정을 구글 캘린더와 함께 관리할 수 있는 개인용 스케줄 관리 서비스를 만든다.

## 타겟 사용자

FLO 직원을 포함해 개인 업무 일정을 체계적으로 관리하고 싶은 사용자

## 주요 기능

### 프로젝트 일정 한눈에 보기
- 설명: 사용자가 여러 달에 걸쳐 진행되는 프로젝트 일정을 한눈에 확인한다.
- 상세: 프로젝트 시작일과 종료일이 길게 이어지는 형태로 보여서 현재 진행 중인 프로젝트를 직관적으로 인지할 수 있어야 한다.

### 주요 일정 확인
- 설명: 프로젝트 중간의 핵심 마일스톤이나 일정 시간을 빠르게 확인한다.
- 상세: 프로젝트 내 주요 일정은 별도로 강조되어 보여야 하며, 사용자는 일정의 날짜와 시간을 바로 확인할 수 있어야 한다.

### 업무 우선순위 지정
- 설명: 사용자가 각 업무나 일정의 우선순위를 구분해서 관리한다.
- 상세: 일정마다 우선순위를 지정할 수 있고, 화면에서 우선순위 차이가 분명하게 드러나야 한다.

### 구글 캘린더 연동
- 설명: 사용자가 선택한 일정을 구글 캘린더에 추가하고 함께 확인한다.
- 상세: 일정의 날짜와 시간이 포함된 상태로 미리 등록한 구글 캘린더에 추가할 수 있어야 하며, 구글 캘린더 일정도 서비스 안에서 함께 열어보거나 확인할 수 있어야 한다.

## 사용 환경

- 플랫폼: 모바일 웹 / PC 웹
- 로그인 필요 여부: 불필요

## 데이터 저장

- Supabase 사용: 미사용
- 저장할 데이터: 없음

## 외부 연동

- 구글 캘린더 일정 추가
- 구글 캘린더 일정 조회 또는 함께 보기
