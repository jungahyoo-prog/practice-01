## 🚀 기능 목록

### [F1] 프로젝트 일정 메인 화면
> **Status**: ✅ 완료
> **목표**: 사용자가 여러 달에 걸친 프로젝트와 진행 중인 업무를 한눈에 확인할 수 있다.

- [x] 메인 화면 구조와 핵심 영역을 설계한다
- [x] 장기간 프로젝트가 이어져 보이는 일정 표시 방식을 만든다
- [x] 진행 중인 프로젝트를 직관적으로 보여주는 시각 요소를 정리한다
- [x] 모바일과 PC 웹에서 모두 보기 좋은 레이아웃으로 다듬는다

---

### [F2] 주요 일정과 우선순위 관리
> **Status**: ✅ 완료
> **목표**: 사용자가 프로젝트 안의 중요한 일정과 업무 우선순위를 빠르게 확인하고 관리할 수 있다.

- [x] 주요 일정과 일반 일정을 구분하는 입력 항목을 만든다
- [x] 일정 날짜와 시간을 쉽게 확인할 수 있는 상세 표시를 만든다
- [x] 업무별 우선순위를 지정하는 방법을 정한다
- [x] 우선순위에 따라 화면에서 차이가 보이도록 디자인한다

---

### [F3] 일정 추가와 수정 흐름
> **Status**: 🔄 진행 중
> **목표**: 사용자가 프로젝트와 주요 일정을 직접 추가하고 수정할 수 있다.

- [ ] 새 프로젝트와 일정을 입력하는 화면 또는 패널을 만든다
- [ ] 프로젝트 기간, 주요 일정, 우선순위를 입력하는 폼을 만든다
- [ ] 작성한 일정이 메인 화면에 바로 반영되도록 연결한다
- [ ] 일정 수정과 삭제 흐름을 정리한다

---

### [F4] 구글 캘린더 연동
> **Status**: ⏳ 예정
> **목표**: 사용자가 선택한 일정을 구글 캘린더에 추가하고 관련 일정을 함께 확인할 수 있다.

- [ ] 구글 캘린더 연동 방식을 정하고 필요한 준비 사항을 확인한다
- [ ] 선택한 일정의 날짜와 시간이 포함된 캘린더 추가 동작을 만든다
- [ ] 구글 캘린더 일정을 함께 확인하는 화면 흐름을 정리한다
- [ ] 모바일과 PC 웹에서 연동 경험이 자연스럽도록 마무리한다
